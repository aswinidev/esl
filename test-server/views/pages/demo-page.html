Smart Image

Smart Image - is the first Smart Library component. It simplify work with images.

Smart Image Features:

  • Smart Query
  • Lazy Loading
  • Native-like API

Doesn't Smart Image Config looks simple?

<smart-image lazy mode="cover"
             data-alt="Alt Text Test"
             data-src="
               img-1-carousel-9-6.jpg |
               @LG => img-2-carousel-9-6.jpg |
               @MD => img-3-carousel-9-6.jpg |
               @SM => img-4-carousel-9-6.jpg |
               @XS => img-5-carousel-9-6.jpg
             "
             data-src-base="/images/"></smart-image>

Smart Media Features:

Smart Media Providers
