Smart Media - is a smart container that gives you the same API for different providers under the hood along with helpers like lazy loading, activate query, picture cropping.

YouTube video

Smart Media has full support of YouTube provider. And you can use smart media component the same way as a native video element

HTML video

Smart Media can be used for simple HTML videos. It brings us the ability to use extra features (like lazy loading or cropping) for simple videos.

Basic Iframe Example

Smart Media also has build-in provider to wrap abstract iframe source. Here is a simple example for Vimeo media source

Basic Iframe provider has limited API so you can only show/hide/reinitialize inner iframe

Basic Audio Example

Smart Media is also support simple HTML audio.

HTML video background

Cropping and decorations features of Smart Media allows us to use it as an active background for the block.
